Compiler for the q programming language.

    Run a source file:

        q [file]

    Build an executable:

        q build [file]

    Build for another system:

        q build [file] --os [linux|mac|windows] --arch [x86|arm]

    Show output from various stages:

        q build [file] --asm      # assembly code
        q build [file] --ssa      # static single assigment form
        q build [file] --verbose  # all of the above

    Show commit date and hash:

        q version
