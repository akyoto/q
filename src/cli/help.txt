Compiler for the Q programming language.

    Run a source file:

        q [file]

    Build an executable:

        q build [file]

    Build options:

        -arch [x86|arm]              # sets the processor architecture
        -os [linux|mac|windows]      # sets the operating system
        -no-lint                     # disables the linter
        -no-optimize                 # disables the optimizer

    Show intermediate stages:

        q asm [file]                 # shows assembly code (accepts build and filter options)
        q ssa [file]                 # shows SSA form (accepts build and filter options)

    Filter options:

        -func [name]                 # filters output by function name

    List all imported files:

        q files [file]               # accepts build options

    Query information:

        q help                       # shows this help
        q keywords                   # lists keywords
        q operators                  # lists operator precedence
        q version                    # shows the version
