Compiler for the q programming language.

    Run a source file:

        q [file]

    Build an executable:

        q build [file]

    Build for another architecture:

        q build [file] --arch [x86|arm]

    Build for another operating system:

        q build [file] --os [linux|mac|windows]

    Show assembly code:

        q build [file] -asm (--assembly)

    Show intermediate representation:

        q build [file] -ir (--intermediate)

    Show output from all stages:

        q build [file] -v (--verbose)

    Show this help:

        q help
