Compiler for the q programming language.

    Run a source file:

        q [file]

    Build an executable:

        q build [file]

    Compile for another architecture:

        q build [file] --arch [x86|arm]

    Compile for another operating system:

        q build [file] --os [linux|mac|windows]

    Show verbose output:

        q build [file] -v
        q build [file] --verbose

    Show this help:

        q help
