Compiler for the q programming language.

    Run a source file:

        q [file]

    Build an executable:

        q build [file]

    Build for another system:

        q build [file] --os [linux|mac|windows] --arch [x86|arm]

    Show output from various stages:

        --asm            # assembly code
        --func [name]    # filter by function name
        --ssa            # static single assigment form
        --verbose        # all stages
