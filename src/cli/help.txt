Compiler for the Q programming language.

    Run a source file:

        q [file]

    Build an executable:

        q build [file]

    Build for another system:

        q build [file] -os [linux|mac|windows] -arch [x86|arm]

    Show all imported files:

        q files [file]

    Show intermediate stages:

        q [asm|ssa] [file] -func [name]
