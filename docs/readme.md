# q

A programming language that quickly compiles to machine code.

## Platforms

| CPU    | Linux | Mac | Windows |
| ------ | ----- | --- | ------- |
| arm64  | ✅    | ❔  | ❔      |
| x86-64 | ✅    | ✅  | ✅      |

Those marked with a ❔ are theoretically supported but there are no developer machines to test them.
